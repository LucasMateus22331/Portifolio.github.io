💼 Portfólio de Estágio em Desenvolvimento

Repositório criado com o objetivo de apresentar projetos, exercícios e práticas desenvolvidas durante minha formação em Desenvolvimento de Software, com foco em aprendizado contínuo e aplicação de conceitos fundamentais da área.

Este portfólio é voltado para processos seletivos de estágio, demonstrando organização, lógica de programação e boas práticas de código.

🎯 Objetivo do Projeto

Demonstrar conhecimentos básicos e intermediários em desenvolvimento

Praticar estruturação de projetos

Aplicar conceitos aprendidos em cursos e estudos pessoais

Evoluir tecnicamente para o mercado de trabalho

🛠️ Tecnologias Utilizadas

Node.js (em processo)
